# bologna-dashboard
A personal dashboard tracking my move to Bologna, Italy — deadlines, contacts, documents, and savings.
Built as a single-file PWA — runs in any browser, installable on Android and desktop.

## Features

- Live Bologna time with Italian greeting
- Countdown to move date
- Important dates and deadlines tracker
- Document checklist
- Contacts
- Quick notes
- Savings buffer tracker
- Google Drive sync

## Setup

1. Clone or download `bologna.html`
2. Open in Chrome
3. Add your Google OAuth Client ID in Settings for Drive sync
4. On mobile: open in Chrome → Add to Home Screen

## Stack

Vanilla HTML/CSS/JS — no dependencies, no build step.
